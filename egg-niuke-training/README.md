## 🎯 学习建议

1. **先理解题意**：仔细读题，理解输入输出
2. **画图分析**：复杂题目画图辅助理解
3. **暴力求解**：先想出暴力解法
4. **优化思路**：考虑时间和空间优化
5. **代码实现**：注意边界条件
6. **测试验证**：多个测试用例验证
